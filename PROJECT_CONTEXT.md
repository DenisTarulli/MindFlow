# MindFlow

Proyecto experimental de Unity 2D.

Objetivo:
Investigar sistemas de dificultad adaptativa en puzles.

Restricciones:
- Sin personaje jugable.
- Sin movimiento.
- Sin exploración.
- Sin combate.
- Sin Machine Learning.

Prioridades:
1. Recolección de métricas.
2. Adaptación dinámica.
3. Modularidad.
4. Exportación de datos para investigación.
